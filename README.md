# Extracteur de Factures PDF (Version corrigée)

## ✅ Instructions pour tester localement
1. Installez Python 3.9+
2. `pip install -r requirements.txt`
3. `streamlit run app_streamlit_easyocr.py`

## ✅ Déploiement sur Streamlit Cloud
- Créez un dépôt GitHub
- Ajoutez ces fichiers
- Déployez en choisissant `app_streamlit_easyocr.py`

## ✅ Améliorations incluses
- Interface améliorée (logo, couleurs, multilingue)
- OCR multi-pages
- Export Excel enrichi
- Gestion des erreurs
